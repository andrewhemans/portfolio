Rainforest Test Designer

Test Designer is a feature I worked on for Rainforest QA which allowed users to enter rough outlines for regression tests that would be converted into fully automatable tests by the Rainforest Crowd.

Project Goals

  • •Make test writing super easy
  • •Allow users to convert existing tests into Rainforest format
  • •Return a batch of tests within 48 hours
